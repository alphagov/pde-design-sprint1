Request to use an eligibility checker

Use this service to:

  • draft a Data Sharing Agreement (DSA) request that will let you use a GOV.UK eligibility checker.
  • submit the DSA request to the government department that owns the data you need.

Requesting takes around 15 minutes. You can review the request before it's sent.

Start now

Before you start

Only central and local government can use this service. You'll need an e-mail address ending with gov.uk to apply.

You'll need to know the name and e-mail address of your department's Senior Information and Risk Owner (SIRO) or equivalent information owner to complete the requset. They'll need to sign the request before it can be used.
